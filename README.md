# 문화재 지도 프로젝트
이 프로젝트는 지도에서 문화재 위치를 확인하고, 마커를 클릭하면 문화재에 대해 상세하게 볼 수 있습니다.

### 활용 기술
- Spring Boot
- Spring Data JPA
- Google Login API
- Kakaomap API
- ETRI 형태소 분석 API
